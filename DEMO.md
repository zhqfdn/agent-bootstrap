## 2. 输入系统 - 意图识别

```
📋 "帮我写一个程序"
   → 意图: task (置信度: 65%)

❓ "今天天气怎么样？"
   → 意图: question (置信度: 95%)

⚡ "/help"
   → 意图: command (置信度: 100%)

💬 "你好啊！"
   → 意图: chat (置信度: 80%)
```

```

【请回复你的名字】或者直接说"开始"使用默认配置
```
